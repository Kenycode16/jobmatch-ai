# JobMatch AI

A Chrome extension that scrapes LinkedIn job listings based on user preferences and saves them for AI-powered resume matching.

## Features
- Search LinkedIn jobs by title, location, experience level, work type, and time posted
- Bulk scrape job listings from search results
- Save jobs to Chrome storage for later analysis
- Background tab scraping (popup stays open with live status)

## Tech Stack
- Chrome Extension (Manifest V3)
- JavaScript
- Chrome APIs (tabs, storage, scripting)

## Installation
1. Clone this repo
2. Open Chrome → `chrome://extensions`
3. Enable Developer Mode
4. Click "Load unpacked" → select the `chrome-extension` folder
5. Pin the extension and start searching!
